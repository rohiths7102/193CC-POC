## Fixed the same day (post-audit hardening, all verified live)

1. **Summit invitations now exist at runtime** — admin/sales invite any member per category from the Summit console (audited + emailed); previously seed-only.
2. **Temporary Membership is truly event-linked** — applicant picks Presentation / Brand Launch / Award at enrolment (live slots-left shown), activation auto-creates the INVITED application and sets access expiry to **event date + 7 days** (was flat 30 days).
3. **"One opportunity, one event, per annum" enforced** — a member cannot hold two live applications across categories of the same Summit.
4. **Admin mentor assignment** — assign/end Mentor & Consultant per member with track (senior/enterprise/investment); previously seed-only.
5. **Mentoring rollover setting is live logic** — `mentoring.rollover_enabled=true` now carries last month's unused hours (capped at +1 month) in the booking allowance check.
6. **Scheduled Direct Debit collections** — the 60s tick now auto-collects due instalments; failures mark the plan **AT_RISK** (audited) and surface in admin.
7. **Admin deal-flow coordination view** — all pitches + every investor expression of interest, consent state visible (was investor-only).
8. **Pitch materials upload** — members attach decks; investors see download links (consent-gated).
9. **News & networking area** — `networking_news` entitlement now opens a real page: published member articles + events calendar (published articles finally render somewhere).
10. **Member database filters** — tier / account status / payment-health (incl. at-risk) filters added to the admin member list.
11. **Video production pipeline carries data** — shoot date captured on scheduling, final asset uploaded on delivery (was status-only).
12. **Membership fees editable in admin Settings** — audited price changes; closes the "configurable, not hard-coded" caveat fully.
